Ativar Firebase Authentication

Clique abaixo para abrir o Firebase Console e ativar os métodos de login:

Abrir Firebase Auth

  1. Faça login no Firebase
  2. Selecione o projeto sistema-apb
  3. Abra E-mail/senha → clique em Ativar → Salvar
  4. Abra Google → clique em Ativar → escolha e-mail → Salvar
