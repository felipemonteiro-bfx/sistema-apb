Configurar credenciais do Firebase

Siga os passos abaixo para corrigir o erro api-key-not-valid.

Passo 1 – Copiar credenciais

Abrir Firebase Console

  1. Selecione o projeto Sistema APB
  2. Role até Seus apps
  3. Clique no app Web (</>)
  4. Copie cada valor do firebaseConfig

Passo 2 – Editar .env.local

Abra o arquivo .env.local na raiz do projeto e cole os valores assim:

VITE_FIREBASE_API_KEY=AIzaSyXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX
VITE_FIREBASE_AUTH_DOMAIN=seu-projeto.firebaseapp.com
VITE_FIREBASE_PROJECT_ID=seu-projeto-id
VITE_FIREBASE_STORAGE_BUCKET=seu-projeto.appspot.com
VITE_FIREBASE_MESSAGING_SENDER_ID=123456789
VITE_FIREBASE_APP_ID=1:123456789:web:abcdef123456
  • Sem aspas nos valores
  • Sem espaços antes ou depois
  • Formato: NOME=valor

Passo 3 – Reiniciar o servidor

No terminal, na pasta do projeto:

npm run dev

Ou execute setup.bat ou configurar-firebase.bat.
