NFS-e Manaus (AM)

Configuração necessária

A emissão automática usa a API da NFE.io. Para ativar:

  1. Cadastre-se em app.nfe.io
  2. Cadastre sua empresa e faça upload do certificado digital A1
  3. Credencie-se na Prefeitura de Manaus (Nota Manaus)
  4. Obtenha a chave de API e o ID da empresa na NFE.io
  5. Configure no Firebase: Functions → emitirNFSe → Variáveis de ambiente → NFE_IO_API_KEY, NFE_IO_COMPANY_ID
