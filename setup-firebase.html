🔥 Setup Firebase

Sistema APB - 5 minutos para começar

1

Acessar Firebase Console

  • Abra: https://console.firebase.google.com
  • Faça login com sua conta Google (crie uma se não tiver)
  • Será grátis! Plano Spark
Abrir Firebase →
2

Criar Novo Projeto

  • Clique + Criar Projeto
  • Nome: sistema-apb
  • Desabilite Google Analytics (não precisa)
  • Clique Criar Projeto
  • ⏳ Aguarde ~1 minuto
3

Ativar Firestore Database

  • No menu lateral esquerdo, clique Firestore Database
  • Clique Criar banco de dados
  • Localização: Escolha a mais próxima (ex: us-central1)
  • Modo de segurança: Modo de teste ✓
  • Clique Criar
⚠️ "Modo de teste" é apenas para desenvolvimento. Depois você configura regras de segurança.
4

Criar Coleções

  • No Firestore, clique + Criar coleção
  • Crie estas coleções (uma por uma):
clientes chapas servicos custos_servico pagamentos notas_fiscais

Deixe vazias (dados criarão via app)

5

Copiar Credenciais

  • Clique ⚙️ Configurações do Projeto (engrenagem no menu)
  • Vá em abas Sua Apps → procure por Web
  • Se não houver, clique + Adicionar app e selecione Web
  • Copie TODA a seção firebaseConfig
const firebaseConfig = { apiKey: "...", authDomain: "...", projectId: "...", storageBucket: "...", messagingSenderId: "...", appId: "..." };
6

Preencher .env.local

  • Abra o arquivo: .env.local (na pasta do projeto)
  • Copie os valores do firebaseConfig para aqui:
VITE_FIREBASE_API_KEY=seu-api-key VITE_FIREBASE_AUTH_DOMAIN=seu-projeto.firebaseapp.com VITE_FIREBASE_PROJECT_ID=seu-project-id VITE_FIREBASE_STORAGE_BUCKET=seu-projeto.appspot.com VITE_FIREBASE_MESSAGING_SENDER_ID=seu-sender-id VITE_FIREBASE_APP_ID=seu-app-id

Salve o arquivo (Ctrl+S)

7

Rodar o Sistema

  • Opção A (Automático): Duplo clique em setup.bat
  • Opção B (Manual): No terminal:
npm install npm run dev
✅ Navegador abrirá em http://localhost:3111
8

Testar Sistema

  • Vá em 🚚 Clientes → + Novo Cliente
  • Cadastre um cliente teste
  • Vá em 👤 Chapas → cadastre uma chapa
  • Vá em 📋 Serviços → crie um serviço
  • Volte em 📊 Dashboard → veja os dados aparecerem!
🎉 Sistema funcionando com Firebase!

⏱️ Tempo total: ~5 minutos
