🚀 PRÓXIMOS PASSOS (3 PASSOS APENAS)
================================================================================

1️⃣ CRIAR BANCO DE DADOS (no Supabase)
   └─ Abra: https://supabase.com
   └─ Execute schema.sql na aba SQL Editor
   └─ Tempo: 2 minutos

2️⃣ CONFIGURAR AMBIENTE
   └─ Copie chaves do Supabase (Settings > API)
   └─ Cole no arquivo: .env.local
   └─ Tempo: 1 minuto

3️⃣ RODAR O SISTEMA
   └─ Duplo clique: setup.bat
   └─ OU: npm install && npm run dev
   └─ Tempo: 3 minutos
